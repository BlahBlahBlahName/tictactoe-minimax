Tic-Tac-Toe Minimax (Qt 6)
A high-performance, unbeatable Tic-Tac-Toe game built with C++ and the Qt 6 Framework. This project features a graphical user interface and an AI opponent powered by the Minimax Algorithm, ensuring the computer never loses.

Built With:

Language: C++17

Framework: Qt 6.9.1

Build System: CMake

Compiler: LLVM-MinGW
